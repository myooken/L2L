# Security Policy

## Supported Versions
`main` ブランチのみをサポート対象とします。

## Reporting a Vulnerability
脆弱性やセキュリティ上の懸念を発見した場合は、**Issue を立てずに** 以下の方法でご連絡ください。

- GitHub の Private Vulnerability Reporting
- もしくは README に記載している連絡先（未記載の場合はリポジトリオーナー宛てに直接ご連絡ください）

調査のために、再現手順・環境・ログ（可能なら最小限）を共有いただけると助かります。

## Notes
- 本プロジェクトは PeerJS を用いた P2P 通信を行います。共有リンクやシグナリング ID を第三者と共有する際は、公開範囲と接続先を慎重に判断してください。
- 回答データはサーバーに保存しませんが、共有 URL に圧縮された回答が含まれます。URL を共有した相手には内容が閲覧可能になります。
- TURN サーバーや PeerJS サーバーの資格情報を公開リポジトリへコミットしないでください。認証情報を扱う場合は .env 系ファイルに隔離し、ログにも残さないようにしてください。
